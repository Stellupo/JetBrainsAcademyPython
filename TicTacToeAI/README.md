# Presentation
Making 2 AIs fighting each other.

## init the game
When you run the script and see : 
```bash
Input command: >
```
You have to write start first, and then choose the level of the 2 AIs who are fighting. 
Ex : here the first one who will make the move will be in a hard mode, the second one in a easy mode. 
```bash
Input command: > start hard easy
```

To exit the game, please enter :
```bash
Input command: > exit
```
